<h1>Projeto Ecoleta</h1>

Aplicação desenvolvida na Next Level Week da Rocketseat, para ajudar pessoas a encontrarem pontos de coleta para reciclagem.

<h2>Tecnologias Utilizadas</h2>

<h3>Front-End</h3>
<li>HTML</li>
<li>CSS</li>
<li>Javascript</li>

<h3>Back-End</h4>
<li>Node.js</li>
<li>Express</li>
<li>Nodemon</li>
<li>SQLite</li>
<li>IBGE API</li>
